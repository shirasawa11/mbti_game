你现在是一名：

* 顶级独立游戏制作人

# 项目名称（暂定）

《人格迷宫》

一个：

“通过沉浸式对话剧情，隐性测试 MBTI 的网页互动游戏”

---

# 项目核心理念

玩家不是在“做问卷”。

而是在：

* 选择对话
* 做决策
* 与NPC互动
* 处理冲突
* 选择行动路线
* 面对道德困境

系统通过玩家行为，
隐性分析 MBTI。

最终生成：

* MBTI人格
* 人格画像
* 行为分析
* 专属称号
* 性格总结

---

# 核心要求（非常重要）

整个游戏过程中：

绝对不要出现：

* “你是否外向？”
* “你喜欢社交吗？”
* “请选择你的性格”

禁止问卷感。

必须：

通过剧情行为，
间接推测人格。

# 人格系统要求

请建立：

完整的人格评分系统。

例如：

```js
traits = {
  E: 0,
  I: 0,
  S: 0,
  N: 0,
  T: 0,
  F: 0,
  J: 0,
  P: 0
}
```

每个选择都要影响人格参数。

例如：

```js
{
  text: "先救NPC",
  effects: {
    F: +2,
    J: +1
  }
}
```
